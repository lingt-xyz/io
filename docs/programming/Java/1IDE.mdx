title: IDE
---

This tutorial demonstrates how to use IDEs to write a Java program on ENCS lab computers. 

## Windows

### 1 Open Eclipse

> Windows &gt; eclipse\_OXYGEN

![](/assets/java/windows_eclipse.png)

### 2 Create a Java project

> File &gt; New &gt; Project

![](/assets/java/windows_eclipse_project.png)

![](/assets/java/windows_eclipse_project_java.png)

Name your project, e.g. :`Hello_World`. Then click _Finish_.

![](/assets/java/windows_eclipse_project_java_create.png)

### 3 Perspective

If you are prompted to choose a perspective, you can click _Open Perspective_.

![](/assets/java/windows_eclipse_perspective_check.png)

If after clicking you cannot see the perspective, you may need to click the up-left button to toggle on the view.

![](/assets/java/windows_eclipse_perspective_show.png)

After clicking you should see the following project we just created.

![](/assets/java/windows_eclipse_project_java_show.png)

### 4 Create a Class

> src \(right click\) &gt; New &gt; Class

![](/assets/java/windows_eclipse_project_java_class.png)

Name your class, e.g.: `Main`. Then click _Finish_.

![](/assets/java/windows_eclipse_project_java_class_name.png)

### 5 The main method

Now we can write the main method in our newly created class.

![](/assets/java/windows_eclipse_project_java_class_method.png)

### 6 Run

![](/assets/java/windows_eclipse_project_java_run.png)

### 7 Hello World

We can see the output of our program in the console.

![](/assets/java/windows_eclipse_project_console.png)
